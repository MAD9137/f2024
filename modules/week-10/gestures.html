# Overview

Gesture modifiers add interactivity to an application allowing developers to include logic that is executed in response to various touch events.

  • Add gesture modifiers to a specific SwiftUI view in the view hierarchy
  • When there are competing gestures, by default a child view's gesture will take precedence over a parent
  • Change precedence with highPriorityGesture() to force a parent's gesture or simultaneousGesture() to combine both gestures
  • Use ExclusiveGesture() to configure which gesture takes precedence between two selected gestures
  • Create a gesture that is a sequence of two gestures using SequenceGesture()

# Gesture Modifiers

# onTapGesture

// Defaults to 1 tap
Text("Hello, world!")
    .onTapGesture {
        print("Tapped on TextView")
    }

// Double-tap
Text("Hello, world!")
    .onTapGesture(count: 2) {
        print("Double-tapped on TextView")
    }

# onLongPress

// Basic implementation
Rectangle()
    .onLongPressGesture {
        print("Long press gesture")
    }

// Advanced
// 5 seconds, 10 points of movement permitted
Rectangle()
    .onLongPressGesture(
        minimumDuration: 5.0,
        maximumDistance: 10.0,

        // executes after minimumDuration has run
        perform: { 
            print("Long press is over!")
        },

        // executes anytime a change is observed
        onPressingChanged: {
            state in print("Being pressed...\(state)")
        }
    )

# Advanced Gesture Modifier

Using the gesture modifier offers additional configuration options and customization including updates from the following three events:

  • updating
  • onChanged
  • onEnded

WARNING

Not all events are available on all gestures. Refer to documentation for details.

# A swipe gesture

Rectangle()
    .fill(.red)
    .gesture(
        DragGesture()
            .onEnded { value in
                
                // swipe must be +- 25 points of origin on horizontal
                if value.translation.height < 25.00 && value.translation.height > -25.00 {
                    
                    // swipe must be at least 15 points long
                    if value.translation.width < -15.00 {
                        print("You swiped right to left")
                    } else if value.translation.width > 15.00{
                        print("You swiped left to right")
                    }
                }
            }
    )
